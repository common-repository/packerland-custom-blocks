This plugin was created to extend the gutenberg editor and make it more user friendly.

== Description ==

Extends Gutenberg to have a better user experience.

Major features include:

* Advanced Facebook Feed Block
* Advanced Paragraph Block
* Full Width Editor

== Screenshots ==
1. Packerland Websites Category (Version 1.0.0)
1. Advanced Paragraph Block (Version 1.0.7)
2. Advanced Paragraph Block Menu (Version 1.0.0)


== Installation ==
1. Upload the entire plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
